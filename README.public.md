# Friday

A personal AI assistant workspace powered by OpenClaw.

This repository contains the configuration, skills, and supporting files used to run and evolve **Friday** — a practical, persistent assistant environment focused on daily productivity and automation.

## What’s inside

- Assistant behavior/config files
- Custom skills and references
- Utility scripts
- Lightweight project/planning artifacts

## Quick start

```bash
npm install
```

git status
